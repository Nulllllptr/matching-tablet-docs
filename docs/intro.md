# マッチングタブレット 導入ガイド

`MatchingGimmick` プレハブをシーンに置けば動きます。あとは**触りたい所だけ**、下の各ページを見て編集してください。

## 編集ページ一覧

| 変えたいこと | ページ |
|---|---|
| ボタン文言・タブ名 | [文言・タブ名](./text-config) |
| 召喚キー・出る位置・投げる強さ・しまう | [召喚・しまう・投げ](./summon) |
| 通知音の素材・音量・種類 | [通知音](./notify) |
| サイズの上限/下限 | [サイズ変更](./size) |
| 顔アイコンの精度・人数・写り | [顔アイコン](./portrait) |
| 行ボタンの色 | [行ボタンの色](./row-color) |
| 起動ヒントの文言・表示秒数 | [起動ヒント](./hint) |
| 召喚できるエリアの限定 | [召喚エリア](./summon-area) |

## 設置時のポイント

- **初期画面**：起動時に `Panel_Home` だけ出したい場合、`Panel_Home` を有効、`Panel_UserList` / `Panel_InterestedInMe` / `Panel_Settings` を**非アクティブ**にしておく。
- 上記以外の参照欄（Data Manager / Menu Controller など）は基本そのままでOKです。
